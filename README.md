# 🏥 Hospital Analysis Report

## 📌 Project Description

=> This report provides a comprehensive analysis of **doctor scheduling, patient treatment timelines, and doctor performance** within a healthcare facility. It is designed to evaluate operational efficiency, patient care quality, and the effectiveness of medical staff in delivering timely, high-quality healthcare services.

---

## 📊 Data Source

=> Dataset is self-created for analysis purposes.

---

## 🛠 Tech Stack

=> Power BI – used for data modeling and visualization.

---

## 🔧 Tasks Performed

Built an interactive report with the following insights:

     => Doctor performance metrics and experience
     => Patient treatment timelines and recovery outcomes
     => Department-level analysis including treatment types, total patients, and total cost

Created measures for analysis:

     => Female Patient – Number of female patients
     => Male Patient – Number of male patients
     => Total Patient – Total number of patients
     => Recover Patient – Number of recovered patients
     => Recover % – Percentage of recovered patients

---

## ✨ Key Features

Cards (used as KPIs):

     => No. of Doctors
     => No. of Specialized Doctors
     => Average Age of Doctors
     => No. of Departments
     => No. of Treatment Types
     => Recover Patient %
     => No. of Patients
     => No. of Male & Female Patients

Cards (used as buttons for navigation):

     => Doctor
     => Department
     => Patient

Slicers:

     => City
     => Admission Date
     => Patient Name
     => Doctor Name
     => Department Name

Charts & Visuals:

=> **Doctor Page**

       * Column Charts: Doctor Name vs Experience, Doctor Name vs Average Fees
       * Matrix: Doctor Name, Outcome, No. of Patients
       * Table: Doctor ID, Doctor Name, Specialization, Count of Patients

=> **Department Page**

       * Column Chart: Department Name vs Experience
       * Pie Chart: Department Name vs Total Fees
       * Table: Department Floor, Department ID, Department Name, Average Age, Total Patients, Sum of Total Cost

=> **Patient Page**

       * Column Chart: Patient ID vs Total Fees
       * Gantt Chart: Patient ID, Admission Date, Departure Date
       * Table: Patient ID, Outcome, Treatment Type, Doctor Name, Insurance

Navigation:

      => Forward buttons for page navigation
      => Bookmark-based open/close menu

## 🎯 Goal
=> To provide insights on doctor performance, patient recovery, and department efficiency, supporting improved decision-making and operational management in healthcare facilities.To provide insights on **doctor performance, patient recovery, and department efficiency**, supporting improved decision-making and operational management in healthcare facilities.

## Insight
     => Approximately 40% of patients are not covered by insurance, indicating a significant gap in healthcare financial coverage.
     => Nearly 50% of patients remain unrecovered during the observed period, suggesting challenges in treatment outcomes or recovery time.
     => More than 25% of patients require cardiology consultation, highlighting a higher prevalence of cardiac-related health issues.
